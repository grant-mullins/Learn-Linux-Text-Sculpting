# sed
The sed (stream editor) command allows you to filter and edit streams of text. By default, it performs each operation on one line at a time.

## Common flags
### -n
Only print lines that match the given regex.
### -e
Used to specify multiple sed commands.
### -i
Edit the file in place instead of printing to stdout.
### -E
Use extended regular expression.
### -z
Separate lines by NUL characters.
